Desktop game of S.K.A.T.E built with Python Tkinter

First substantial project

This is NOT a skateboarding simulation game, but rather a recreation of the "game of S.K.A.T.E." typically played in person with another player. This recreation is a project made from an idea of catering to fingerboard games of S.K.A.T.E., hence being able to be played by yourself against a bot.

Made with Python 3.14.5
